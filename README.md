# HTU21D_SoftI2C
This is an Arduino library for HTU21D, Si7021 and SHT21 Digital Humidity & Temperature Sensor

Supports all sensors features:

- read Humidity
- calculate Compensated Humidity
- read Temperature
- change Resolution
- soft Reset
- check Battery Status
- turn ON/OFF build-in Heater
